## 마트 계산 시스템

### 마트 계산 프로세스
- 고객이 상품을 카트에 담는다.
- 마트오너는 계산하기 전 지불 정보를 만든다.
- 고객은 지불 정보를 바탕으로 결제한다.(결제 수단은 오직 카드뿐이다.)
